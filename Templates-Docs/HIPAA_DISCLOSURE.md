# How to share the HIPAA Disclosure info with the Consulting team


## The HIPAA disclosure must include the patients MRNs and full names, time-frame of data request, date of data delivery and data elements provisioned. 
The Box sub-folder should follow this naming convention: ”IRB# - PI Name - #" If there is more than one request with the same IRB# and PI Name, add a number at the end. For instance, the first one will have no number and the next one will have a “2” and so forth.

    Example: “23-001007 - Bulut” for the first one and “23-001007 - Bulut - 2”




Once the request has been completed, we shall copy the following files to the Box folder called “DataRequest“ and into the corresponding project sub-folder: 


* Data Dictionary: it contains time-frame of data request, date of data delivery and data elements provisioned.

* HIPAA file specifications:
  - txt file
  - formatted with semicolon as delimiter and double quotes for text identifier
  - headers: MRN, last name, first name
  - zipped (using zip)


After copying the files, the programmer shall type the following message in Monday.com. 

---
*Hi, @consultant.*

*(I’ve delivered the HIPAA disclosure file and data dictionary for this project to the corresponding Box project folder.*

*The password to unzip the HIPAA file is XXXXXXXX*

*Please let me know if you have any questions.*

*Thank you*

*PS: remember to delete the HIPAA file once the disclosure has been completed. We maintain a backup copy in case it’s needed.*

---

> The project sub-folder name in Box will be stored in Monday.com under [Box Folder].
